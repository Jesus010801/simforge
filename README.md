# simforge
SimForge is an agentic scientific-computing platform for molecular simulation studies.
## Features

- Automatic biomolecular system classification
- Protein and ligand validation
- Workflow generation from YAML specifications
- Adaptive error diagnosis and remediation
- Scientific trajectory analysis
- Checkpointing and workflow recovery

## Supported Systems

- Protein
- Protein–Ligand
- Competitive Inhibition
- Membrane Proteins

## Current Status

- 500+ automated tests
- Active development
- GROMACS-based workflows
